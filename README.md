# 🎯 StreamSniper

**StreamSniper** — это интерактивный киберспортивный веб-прототип игры на реакцию и прогнозирование. Игроки смотрят трансляции соревнований (CS:GO, Футбол, Dota 2) и должны предугадать, когда произойдет следующее ключевое событие (убийство, гол, эпичный ульт и т.д.). 

## 🎮 Механика игры

В отличие от классических игр на реакцию, в StreamSniper вы должны действовать на опережение:
1. **Предугадай (Сделай предикт)**: Оцените макро- и микро-ситуацию на виртуальном стриме и нажмите кнопку **ДО того**, как произойдет экшен.
2. **Отклонение в секундах**: Как только ожидаемое событие происходит, система высчитывает разницу между вашим кликом и моментом события вплоть до сотых долей секунды.
3. **Наградная система**: Чем меньше таймаут (то есть чем ближе был ваш клик к самому событию), тем больше очков вы зарабатываете. 
4. **Ограничения**: Прогноз сгорает, если вы нажали кнопку, но в течение ближайших 8 секунд на стриме так ничего и не произошло.
5. **Турнирная таблица (Leaderboard)**: Зарабатывайте очки и поднимайтесь в лайв-рейтинге (в рамках прототипа реализовано через умную симуляцию ботов и зрителей).

## ✨ Фичи и дизайн
- **Glassmorphism & Неон**: Премиальный, современный UI-дизайн с глубокими тенями, полупрозрачными интерфейсными панелями и пульсирующими неоновыми свечениями.
- **Индивидуальные анимации категорий**: 
  - *CS:GO / Тактика*: Круговой радар-сканер с переливами.
  - *Футбол*: Динамическое футбольное поле и движущийся мяч.
  - *Dota 2 / MOBA*: Переливающиеся руны и магические пульсации.
- **Автономность**: Математический просчет отклонения и анимированное добавление очков к счету без необходимости иметь Backend-базу.

## 🚀 Как запустить

Поскольку этот проект является фронтенд-прототипом, он не требует ни установки зависимостей (Node.js/npm), ни локального сервера.

1. Склонируйте этот репозиторий к себе на ПК:
   ```bash
   git clone https://github.com/ВАШ_НИКНЕЙМ/РЕПОЗИТОРИЙ.git
   ```
2. Перейдите в папку проекта.
3. Просто откройте файл `index.html` двойным кликом (в Google Chrome, Safari, Firefox и др.).
4. Делайте прогнозы и собирайте очки!

*Также вы можете задеплоить проект в один клик через **GitHub Pages**, Vercel или Netlify.*

## 📂 Структура проекта
* `index.html` — Основной макет интерфейса, семантическая разметка окна стрима, панели управления и богборда.
* `style.css` — Стилизация, CSS-переменные (цветовая тема), keyframes для всех эффектов и адаптивная верстка (CSS Grid + Flexbox).
* `app.js` — Core-логика (управление состояниями трансляции, математика расчета отклонений задержки, симуляция "живых" пользователей).
